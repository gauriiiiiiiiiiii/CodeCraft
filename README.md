# Codecraft

A lightweight browser-based code playground with remote execution, snippet sharing, and developer profiles.

## Highlights
- Monaco editor with themes, font sizing, and per-language defaults
- Remote execution via Judge0 with clear stdout/stderr handling
- Snippets with comments, stars, and profile stats
- Clerk auth + Convex data layer

## Quick Start
```bash
npm install
npx convex dev
npm run dev
```
